Environmindful - Climate Change Web Application

Web application used to find information regarding climate change. User interface displays Air Quality Info, energy production levels, and recent new articles in major US cities on DOM.

Retrieves data from multiple APIs using AJAX requests:

  • Google Maps API - Google Maps
  • News API - News Data
  • World Air Quality API - Air Quality Index Data
  • Weather API - Current Weather Data
  • Carma API - Energy Production Data

Note-Taking Web Application

About FlexNotes

User-friendly Note-taking Web Application. Web app features consisting of playable videos, a functional text-editor, and a lecture slides viewer.

User-interface is organized by a dynamic navigation bar that allows the user to organize study materials through a system of binders, tabs, and pages.

Built with ReactJS, Redux, NodeJS, Express, and MongoDB. Utilized Axios network requests to send and retrieve data. Login was handled through Google OAuth.
